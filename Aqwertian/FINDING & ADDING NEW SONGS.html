Finding & Adding New Songs


Aqwertyan lets you play any MIDI (.mid) or KAR (.kar) file! These files can be found all over the internet, and there are millions available. You can import songs into the app in a few different ways:


Using the Store

The store provides two ways to search for song files: a quick search (the Search tab), and a web search (the Web tab). The quick search will automatically search a small selection of MIDI sites, and display the results in the app. For a wider variety, and to find the best quality songs, we recommend using the web search. Tapping on the Web tab will open a browser you can use to download song files. A list of suggested MIDI sites are present when you open the web search, and you can also simply use Google (if using Google, add ".mid" or "midi" to the end of your search query). Aqwertyan will intercept any MIDI files that you download and add them to the store for you! After you close the web browser, any songs you've downloaded will show up in the "My iPad" tab.


iTunes File Transfer

If you have MIDI files already on your computer, you can use iTunes file sharing to add them to Aqwertyan. Once you've shared the files in iTunes, they will show up in the "My iPad" tab in the store.


For an explanation on transferring MIDI files with iTunes, take a look at Apple's step-by-step guide here: http://support.apple.com/kb/ht4094


Emailing Songs

You can also send songs via email. If you attach a MIDI or KAR file to an email, you can tap on the attachment to open it in Aqwertyan. You'll be given the option to translate it immediately, or add it to the store for later consideration. You can send songs to your friends by tapping the "Share" button in the Library!
